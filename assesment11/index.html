que1 . What is CSS an why use it

CSS stands for cascading style sheet . its basically used to create good looking websites by describing how the html elements are displayed on the screen . it also controls the layout of the websites
why css -
CSS is a core technology in web development.
CSS specifies the presentation of HTML documents.
This includes colors, backgrounds, fonts, spacing, animations, etc.
It works on all devices: desktop, tablet, mobile, etc.
CSS is lightweight and relatively easy to learn.
CSS along with HTML are essential skills for web developers.

que2.What are the different ways to bring CSS into an HTML file

there are three ways to bring css in html file

  1. inline css
  2. internal css
  3. external css
inline css -
inline css allows u to add styling to html elements by using style attribute it is called inline because the style is applied directly to the elements in the same line of html code
inline sss has highest specificity means that they will override any other styles that are applied to the same element
internal css -
internal css is using style element in the head of html documents,to define the styles to to the elements in the pages u can apply the style to the elements using css selectors
external css -
to use external css u have to create css file and then u have to applu styles to elemts in that file and then u have to link that file to html document in head tag using link element having rel attribute with a value of stylesheet and href
